## Synonym Detection

4. You've recently noticed that when users search for data science, it returns different results than if a user were to search for data analytics. You and your team have decided that if a user searches either term, **you want both results lists to display**. What can you do so that a user will see results for data analytics or data science whenever they search either term?

<details>

<summary>Hint</summary> 

- Use the search rewrite Synonym Detection
</details>

<details>

<summary>Solution Steps</summary> 

1. Search `data analytics` and click the **blue plus icon** to add a rewrite. If you don't immediately see the icon, hover your mouse over the search box.  

2. From the list of search rewrite options, select **Synonym**  

3. Choose the direction of **symmetric**  

4. Enter `data science` in the Synonym Mappings field  

5. Click the **Save** button  

>The action should immediately take place and a confirmation dialogue box should appear at the top of the screen indicating the rewrite has fired. 

</details>

</br>
